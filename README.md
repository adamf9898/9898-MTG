# 9898-MTG League Web App

A Magic The Gathering (MTG) League web application for a chaotic RPG format. Features tabbed navigation, MTG-themed styles, table of contents, and activities like drafting and deck building. Built with HTML, CSS, and JavaScript only.

## Features
- Tabbed interface for custom MTG relations (objects, variables, functions, etc.)
- MTG-themed styles and assets
- Table of contents for easy navigation
- Sections for drafting, deck building, and league management
- Responsive and accessible design
- Placeholder data and demo content
- Extensible code structure
- Debugging helpers and documentation

## Getting Started
Open `index.html` in your browser. No build step required.

## Folder Structure
- `index.html` - Main entry point
- `css/` - Stylesheets
- `js/` - JavaScript files
- `assets/` - Images and other assets
